Understanding HTML Tables

By: Cam Riemensnider

HTML tables are extremely useful for organizing data on a website. Tables can seem complex at first, but with a little work, you'll pick them up in no time!

Relevant Tags for HTML Tables:

  • <table>
  • <tr>
  • <td>
  • <thead>
  • <tbody>
  • <tfoot>

The above tags are the necessary components that make up HTML table tags. To create the table, you must first use the <table> tag to initialize a table. As with other HTML tags, the <table> tag is not self-closing, so you must nest your table in </table> for it to render properly. Within the <table> element, you must start by adding a table row. The appropriate tag for a table row is <tr>. As with other HTML tags, it must also be closed with an end tag.

Table Tags and Functions

Tag Name Function
<table> Table Initializes the table so you can begin construction
<thead> Table Head Defines the "top" of the table region
<tbody> Table Body Defines the main region of the table
<tr> Table Row Creates a new table row
<td> Table Data Moves across the existing table row and inputs data in each cell
<tfoot> Table Footer Defines the bottom region of the table. Particularly useful for totals.
